# Metaplex Bubblegum (Compressed NFT)
[Metaplex Bubblegum](https://developers.metaplex.com/bubblegum) example codes.

## Setup
```
% cp .env.example .env
```

Then, write your Helius API in .env.

## Run
```
% npm i
% ts-node src/<TS_FILE>
```
